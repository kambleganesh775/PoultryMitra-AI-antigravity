# PoultryMitra AI: Empowering Indian Poultry Farmers

## Problem Statement
Indian poultry farmers face immense challenges including widespread disease outbreaks, high feed costs, and a lack of easy-to-use regional advisory systems. The gap in early disease diagnosis and optimal resource management results in high mortality rates and reduced farm profits. 

## The Solution
poultrymitra-ai---indian-farm-manager bridges this gap by providing an intuitive, localized platform. With an AI-driven smart scanner for spotting diseases, integrated health and invoice management, and multi-language AI chat for accessible solutions (Google Gemini AI), we aim to empower farmers to achieve a productive and profitable farming experience.

## Key Features
- **Smart Scanner**: AI-driven analysis of poultry health from images.
- **AI Expert Chat**: In-context conversational support in English, Hindi, and Marathi.
- **Offline Capable Local DB**: Full support via local browser storage, ensuring it works even in low-connectivity areas.
- **Resource & Chicks Management**: Keep track of everything essential to your farm's success.
- **Sales & Analytics**: Data-driven insights to maximize profitability.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `VITE_GEMINI_API_KEY` in `.env` to your Gemini API key.
3. Run the app:
   `npm run dev`
